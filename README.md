Project Name: 3D Tetris

Description
A 3D version of the classic Tetris game built in Unity, focusing on camera control, piece physics, and spatial alignment.

Tech Stack
 - Unity
 - C#

Current Features
 - 3D camera movement
 - Falling piece mechanics
 - Basic grid logic

Status
 - Work in progress. Future work includes rotation logic, collision detection, and scoring.

Current Controls:
  z/c - Rotate camera
  w/s - Rotate on the horizontal plane
  Left Shift - Speed up current piece falling
  Space - Immediately drop current piece
